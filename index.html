About Me

I build tools that make degen communities safer and smarter.

As a full-stack developer, I specialize in creating Discord-first applications that solve real problems. My flagship project, TiltCheck, is a modular ecosystem of 10+ interconnected tools that help online gambling communities stay safe—from scam detection to responsible gambling features.

What sets me apart? I'm not just building features—I'm building trust systems. Using TypeScript, Discord.js, Supabase, and Cloudflare Workers, I create non-custodial, serverless solutions that put user safety first without compromising functionality.
